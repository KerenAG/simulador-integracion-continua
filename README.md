# Simulador de Integración Continua

## Descripción

Este proyecto consiste en un simulador visual del proceso de Integración Continua (CI), mostrando las etapas principales desde el registro de cambios en el repositorio hasta el despliegue automático en Surge.

El objetivo es demostrar el funcionamiento de GitHub Actions como herramienta de automatización y su integración con un servicio de despliegue continuo.

---

## Objetivo Académico

Implementar una página web estática y configurar un flujo de Integración Continua que permita:

- Subir cambios al repositorio.
- Ejecutar automáticamente un workflow de GitHub Actions.
- Desplegar el proyecto en Surge.sh sin intervención manual.

---

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub
- GitHub Actions
- Surge.sh

---

## Flujo de Integración Continua

1. Se realiza un commit y push al repositorio.
2. GitHub detecta el cambio.
3. Se ejecuta automáticamente el workflow definido en GitHub Actions.
4. El proyecto se despliega automáticamente en Surge.
